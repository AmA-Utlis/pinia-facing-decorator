# 目录功能说明

### 主目录

```shell
├── Template-------------------------------------项目目录
├── src------------------------------------------库文件
├── .browserslistrc------------------------------运行环境配置
├── .editorconfig--------------------------------编辑器规则配置
├── .fatherrc------------------------------------fatherrc配置
├── .gitignore-----------------------------------gitignore配置
├── .npmignore-----------------------------------npmignore配置
├── LICENSE--------------------------------------协议文件
├── package.json---------------------------------库使用的依赖package.json
├── READ_CATALOGUE.md----------------------------目录功能说明
├── READ_CHANGELOG.md----------------------------更新日志说明
├── READ_IDEAR.md--------------------------------临时想法[在开发过程中想到的一些要完善或者要增加功能等等]
├── README.md------------------------------------项目说明
└── tsconfig.json--------------------------------ts配置

```

### src库文件目录

```shell
├── src------------------------------------------项目目录
├── fatherlib------------------------------------库文件
├── types----------------------------------------库文件接口定义
├── utils----------------------------------------库文件方法集合
└── add.ts-------------------------------------库文件入口
```
